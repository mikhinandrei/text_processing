# Text_processing
=================
## Описание
Программа принимает на вход слово на английском языке и ищет похожие слова.
На выходе самое похожее слово.

Для начала нужна база, сгенерированная скриптом `db_creator.py`.
<br/>
Затем можно подать на вход слово `ylon`
И получить на выходе. <br/>
`Maybe you wanted to write  nylon 1`, <br/>
Где последняя цифра - [расстояние Левенштейна](https://ru.wikipedia.org/wiki/Расстояние_Левенштейна) между входным и выходным словами.

## Алгоритм
Входное слово раскладывается по базису, состоящему из букв английского алфавита.
Затем происходит обработка случая, если пользователь пропустил одну из букв (ко всем "координатам" добавляются по 1)
Далее происходит тупой поиск минимального расстояния Левенштейна.

В перспективе считать расстояния между исходным вектором и остальными векторами в базе
и выбирать данные для измерения расстояния в рамках какой-то погрешности.
